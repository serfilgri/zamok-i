Как это работает

Структура страницы

Страница состоит из трёх основных частей: шапка (header), основной контент и подвал (footer). Шапка и подвал хранятся в отдельных файлах в папке includes/.

Сборка

При запуске npm run build:pages скрипт собирает все страницы, подставляя содержимое include-файлов в нужные места.

Обновление

Чтобы изменить навигацию или подвал на всём сайте, достаточно отредактировать один файл в папке includes/ и запустить сборку.
