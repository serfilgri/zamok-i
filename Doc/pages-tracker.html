Pages Tracker

Здесь можно открыть pages-tracker.csv как нормальную таблицу, редактировать статусы, заметки и отметки готовности, а потом сохранить файл обратно. Лучше всего использовать в Chrome или Edge, тогда сохранение пойдёт прямо в исходный CSV.

Файл ещё не загружен. Нажми «Загрузить Doc/pages-tracker.csv» или «Открыть CSV».
Поля status, last_reviewed, ready_100, index_google, index_yandex и notes можно править прямо в таблице.

Быстрые действия

Из обычной HTML-страницы нельзя честно запустить локальную команду в VS Code или терминале. Поэтому я сделал самый простой безопасный вариант: открытие нужных файлов в VS Code и кнопки копирования готовых команд.

Позиции Google

Для проверки нужен не обычный API key, а JSON-ключ service account. У каждой страницы можно задать свой запрос в колонке target_query, а результаты будут записаны прямо в tracker.
Если Google Cloud недоступен, можно экспортировать отчёт из Search Console в CSV и загрузить его здесь.
