📚 ЧТО ТАКОЕ .OPENCODE

веб-сайтов на HTML + Tailwind CSS.

Версия: 1.0.0
Создана: 2026-05-12
Язык: Русский
Фокус: Локальный бизнес SEO

🎯 КЛЮЧЕВЫЕ ПРАВИЛА (из memory.md)
═══════════════════════════════════════════════════════════════

✅ HTML & CSS:
   • Всегда сохранять HTML структуру
   • Никогда не удалять Tailwind классы
   • Поддерживать responsive дизайн
   • Использовать семантический HTML5

✅ SEO:
   • Один H1 на страницу
   • Правильная иерархия заголовков
   • Мета-теги: title (50-60 символов), description (150-160)
   • Оптимизация изображений с alt текстом
   • Естественное использование ключевых слов

✅ Контент:
   • Избегать AI-подобных фраз
   • Писать естественным русским языком
   • Фокус на ценности для пользователя
   • Следовать Google Helpful Content принципам

✅ Локальное SEO:
   • Включать город/регион естественно
   • Оптимизация для "рядом со мной" запросов
   • Использовать локальную терминологию
   • Демонстрировать локальную экспертизу

📊 SEO ЧЕКЛИСТ (основные пункты)
═══════════════════════════════════════════════════════════════

Page-Level SEO:
□ Уникальный title (50-60 символов)
□ Уникальный description (150-160 символов)
□ Один H1 на страницу
□ Правильная иерархия заголовков
□ Минимум 300 слов контента
□ Естественное использование ключевых слов

Technical SEO:
□ Семантический HTML5
□ Alt текст для всех изображений
□ Оптимизированные размеры изображений
□ WebP формат
□ Responsive дизайн
□ Быстрая загрузка (<3 сек)

Local SEO:
□ Город/регион в title и description
□ NAP (Name, Address, Phone) консистентность
□ LocalBusiness schema markup
□ Локальный контент
□ Локальные ключевые слова

Content Quality:
□ Оригинальный контент
□ Без AI-подобных фраз
□ Экспертный уровень написания
□ Ценность для пользователя
□ Google Helpful Content соответствие

🔧 УСТРАНЕНИЕ ПРОБЛЕМ

Проблема: AI удаляет Tailwind классы
Решение: Напомнить "Сохрани все Tailwind классы"
         Сослаться на memory.md правила

Проблема: Контент звучит как AI
Решение: Использовать seo-rewrite.md промпт
         Проверить memory.md правила контента

Проблема: SEO не оптимизировано
Решение: Проверить seo/checklist.md
         Применить seo-rewrite.md промпт

Проблема: Изображения не оптимизированы
Решение: Следовать images/naming-convention.md
         Проверить размеры и форматы
