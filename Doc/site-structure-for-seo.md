## 4) Приоритет на SEO-доработку
1. Приоритет A (в первую очередь): главная + `services.html` + ключевые коммерческие (замена/вскрытие/ремонт/установка) + районные страницы.
2. Приоритет B: страницы доверия и конверсии (`pricing`, `process`, `faq`, `about`, `contact`).
3. Приоритет C: юридические страницы (`offer`, `policy`) — минимально, без переспама.

## 5) Шаблон задания SEO-копирайтеру (можно копировать)
Для каждой страницы из списка:
1. Собрать семантическое ядро: 1 основной кластер + 2-4 вспомогательных.
2. Обновить `Title` (до 60 символов), `Description` (до 160), `H1` (один на страницу).
3. Подготовить SEO-блок 2500-4500 знаков без воды:
   - что делаем;
   - когда нужна услуга;
   - этапы работы;
   - цены/факторы стоимости;
   - гарантии;
   - FAQ (3-5 вопросов);
   - CTA.
4. Добавить LSI-фразы, но без переспама (естественная плотность, без повторов в каждом абзаце).
5. Прописать внутренние ссылки:
   - на родительскую страницу раздела;
   - на 2-4 релевантные смежные услуги;
   - на `pricing` и `contact`.
6. Сохранить уникальность текста между похожими страницами (особенно районными) не ниже 85-90%.
7. Проверить соответствие интенту:
   - коммерческий интент на услугах;
   - информационный интент на FAQ/процессе.

## 📌 SEO-ПРАВИЛА:

1. НЕ повторять текст страницы (писать как дополнение)
2. Использовать LSI-ключи (естественно)
3. Не переспамливать ключ
4. Учитывать интент:
   - услуга → коммерческий
   - FAQ → информационный
5. Добавлять доверие:
   - опыт
   - гарантия
   - срочность
