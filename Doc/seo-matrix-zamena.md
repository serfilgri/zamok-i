# SEO-матрица кластера «Замена» (без каннибализации)

## Базовое правило
- Главная страница `/index.html` — единственная страница под широкий запрос: **«замена замков в Санкт-Петербурге»**.
- Посадочные ниже продвигаются только по узким интентам и не дублируют основной ключ главной в `Title/H1`.

## Правила внутренней перелинковки
- С `services.html` дать отдельные карточки на все 5 URL.
- На каждой посадочной добавить блок «Смотрите также» с 2–3 соседними страницами кластера.
- С первой посадочной (`.../zamena-zamka-vhodnoy-dveri.html`) обязательно ссылаться на:
  - `/service/zamena/zamena-lichinki-zamka.html`
  - `/service/zamena/zamena-zamka-posle-vzloma.html`

## Контроль каннибализации перед публикацией
- `Title` и `H1` у всех 5 страниц уникальные.
- В каждом тексте один главный интент, без смешения нескольких страниц в один кластер.
- Главная страница не получает обратных «конкурирующих» анкоров из этих страниц (избегать анкора в точной форме главного запроса).
