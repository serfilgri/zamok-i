# АНАЛИЗ КОНКУРЕНТОВ - ЗАМЕНА ЗАМКОВ СПБ

## Целевые запросы для анализа

### Высокочастотные (ВЧ)
- замена замков спб (частотность ~5000-10000/мес)
- замена замков санкт-петербург
- вскрытие замков спб
- ремонт замков спб

### Среднечастотные (СЧ)
- замена замка входной двери спб (~1000-3000/мес)
- замена дверных замков спб
- срочная замена замков спб
- замена замка цена спб

### Низкочастотные (НЧ)
- замена цилиндрового замка спб (~100-500/мес)
- замена сувальдного замка спб
- замена замка в металлической двери спб
- вызвать мастера по замкам спб

## ТИПИЧНЫЕ КОНКУРЕНТЫ В ТОП-10 (на основе анализа рынка)

### 1. Крупные агрегаторы услуг
- **Тип:** Яндекс.Услуги, Профи.ру, Юду
- **Преимущества:** Высокий DA/PA, много отзывов, агрегация мастеров
- **Слабости:** Общий контент, нет специализации

### 2. Локальные службы замков
- **Тип:** Специализированные сайты по замкам
- **Примеры доменов:** zamok-service.ru, zamok24.spb.ru, master-zamkov.ru
- **Преимущества:** Узкая специализация, локальный контент
- **Слабости:** Меньший бюджет на продвижение

### 3. Справочники и каталоги
- **Тип:** 2GIS, Yell.ru, Zoon.ru
- **Преимущества:** Высокий трафик, много отзывов
- **Слабости:** Низкая конверсия, общая информация
