2. **Дублирование страниц:**

   - Использовать функцию "Duplicate Page" в Pinegrow
   - Сохранить с новым именем в нужную папку
   - Pinegrow автоматически обновит относительные ссылки

3. **Передача на оптимизацию:**
   - Создать список созданных страниц
   - Передать мне для SEO-оптимизации

### Мои действия (SEO-оптимизация):

Для каждой страницы я оптимизирую:

1. **Meta-теги:**
   - Title (уникальный, с ключевым словом)
   - Description (уникальный, 150-160 символов)
   - Keywords (если используется)
   - Canonical URL

2. **Open Graph:**
   - og:title
   - og:description
   - og:url
   - og:image (подобрать подходящее)

3. **Контент:**
   - H1 (главный заголовок с ключом)
   - H2, H3 (подзаголовки)
   - Основной текст (уникальный, 1500-2500 слов)
   - FAQ (5-10 вопросов)
   - Призывы к действию

4. **Schema.org:**
   - LocalBusiness
   - Service
   - Offer
   - FAQPage
   - BreadcrumbList

5. **Внутренние ссылки:**
   - Ссылки на похожие услуги
   - Ссылки на районы
   - Ссылки на главную и каталог
