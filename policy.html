конфиденциальности

Документ определяет порядок обработки и хранения персональных данных, получаемых через сайт zamok-i.

Основные положения политики

1. Общие положения

Настоящая политика применяется ко всем данным, которые сайт может получить о пользователе при использовании форм обратной связи и других сервисов сайта.

2. Состав данных

Мы можем обрабатывать: имя, телефон, email, текст обращения, а также технические данные (IP-адрес, данные браузера, время обращения).

3. Цели обработки

Данные используются для обработки заявок, связи с пользователем, оказания услуг, улучшения качества сервиса и соблюдения требований законодательства.

4. Передача данных третьим лицам

Передача возможна только в объеме, необходимом для исполнения заявки, либо в случаях, предусмотренных законом.

5. Права пользователя

Пользователь вправе запросить уточнение, обновление или удаление персональных данных. Для этого свяжитесь с нами по телефону или email, указанным в разделе «Контакты».

6. Контакты оператора

По вопросам обработки данных: info@zamok-i.ru, +7 (921) 948-81-72.

Обработка данных

Как мы защищаем информацию

Мы принимаем все необходимые меры для защиты персональных данных пользователей. Данные хранятся на защищённых серверах с ограниченным доступом. Доступ к информации имеют только уполномоченные сотрудники, непосредственно участвующие в обработке заявок.

Персональные данные используются исключительно для связи с пользователем, обработки заказов и улучшения качества сервиса. Мы не передаём данные третьим лицам за исключением случаев, необходимых для исполнения заявки (например, передача адреса мастеру для выезда).

Пользователь может в любой момент отозвать согласие на обработку персональных данных, направив запрос на электронную почту info@zamok-i.ru. В течение 30 дней данные будут удалены из наших систем, за исключением информации, которую мы обязаны хранить согласно законодательству РФ.

Права пользователя

  • Удаление данных по запросу
  • Отзыв согласия на обработку
  • Защиту прав через суд
  • Получение копии данных
