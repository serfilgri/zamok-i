## ⚠️ Возможные проблемы и решения

### Проблема 1: Сборка не работает
**Решение:**
```bash
# Проверьте Node.js версии
node --version  # должно быть >= 14

# Переустановите зависимости
rm -rf node_modules
npm install

# Проверьте наличие шаблонов
ls -la includes/
```

---

### Проблема 2: Страницы не обновляются
**Решение:**
```bash
# Очистите кэш
rm -rf node_modules/.cache

# Запустите с флагом force
npm run build:pages -- --force
```

---

### Проблема 3: Стили "поехали"
**Решение:**
```bash
# Пересоберите CSS
npm run build:css

# Проверьте пути в includes/header.html
# Убедитесь, что пути относительные
```

---

### Проблема 4: Git показывает много изменений
**Это нормально!** После первого запуска изменятся 30+ файлов.

**Что делать:**
```bash
# Просмотрите изменения
git status
git diff --stat

# Если всё корректно — закоммитьте
git add .
git commit -m "refactor: внедрение системы include-файлов

- Созданы шаблоны header, footer, sections
- Удалено 4500+ строк дублей
- Уникализирован контент для service-страниц
- Добавлены рабочие ссылки на соцсети"
```
