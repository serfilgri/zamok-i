**Запросите индексирование для каждой страницы:**

4. `https://zamok-i.ru/process.html`
5. `https://zamok-i.ru/contact.html`
6. `https://zamok-i.ru/service/zamena/zamena-zamka-dveri.html`

**Как запросить:**
1. Введите URL в поле проверки

4. Повторите для каждого URL

### Шаг 2: Sitemap (через 1 день)

1. Перейдите в раздел **"Файлы Sitemap"**
2. Введите: `sitemap.xml`
3. Нажмите **"Отправить"**

---

### Шаг 3: Мониторинг (ежедневно)

1. **Раздел "Покрытие"** → проверяйте:
   - ✅ "Проиндексировано" (должно расти)
   - ⚠️ "Обнаружена, не проиндексирована" (должно уменьшаться)

2. **Раздел "Страницы"** → проверяйте:
   - Количество проиндексированных страниц
   - Ошибки сканирования

## ⚠️ ВОЗМОЖНЫЕ ПРОБЛЕМЫ

### "Обнаружена, не проиндексирована"
**Причина:** Google ещё не сканировал страницу  
**Решение:** Подождать 3-7 дней, запросить индексирование повторно

### "Дубликат, выбранный пользователем не канонический"
**Причина:** Страница имеет 301-редирект  
**Решение:** Нормально, Google должен выбрать каноническую

### "Страница с перенаправлением"
**Причина:** Настроен 301-редирект  
**Решение:** Нормально, проверяйте целевую страницу
